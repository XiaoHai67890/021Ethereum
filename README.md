# 《021 Ethereum》
**想学以太坊，却找不到一本合适的书？**

❌ **《精通以太坊》**：太厚、术语太深，没时间啃，容易劝退。
❌ **《Web3实习手册》**：范围太广，对以太坊底层的讲解不够系统。

对于**小白、学生**以及**Web2 转 Web3 的开发者**，你们需要一个“折中”的方案。

**于是，我写了《021以太坊》。**

📚 **它有什么不同？**

✅ **说人话，不无聊**
拒绝生硬的术语堆砌。我们用**小故事和类比**，把技术讲得像小说一样通俗易懂。

✅ **系统化，重实战**
不仅有专业术语汇总，更是一套完整的工程化学习框架。

✅ **开源共建 (GitHub)**
以太坊在不断升级，教材也会更新。它将作为**下一期实习计划和高校行**的核心教材。

**与其在碎片化信息中迷路，不如从这里开始系统进阶。**
